# Connect Four Game

A Connect Four game built as a solution to the [Frontend Mentor Connect Four challenge](https://www.frontendmentor.io/challenges/connect-four-game-6G8QVH923s).

## Features

- **Player vs Player** - Two players take turns on the same device
- **Player vs CPU** - Play against an AI opponent using minimax with alpha-beta pruning
- **Responsive design** - Works on desktop, tablet, and mobile
- **Animated gameplay** - Smooth piece drop animations and hover effects

## Tech Stack

- TypeScript
- Vite
- CSS (no frameworks)

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Run tests
pnpm test

# Build for production
pnpm build
```
